## Project Overview

MusicBox Roulette is a front-end risk game where users can use coins to roll the slot machine and try to get a match. You play the game until the game ends.
<img width="1017" height="722" alt="Image" src="https://github.com/user-attachments/assets/33f982f7-9416-46e8-a930-d39af9e11dab" />

## Features

- Responsive game that responds to user decisions.
- Application lets user know how many coins they have left and when they get a match or lose.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Musicbox_Roulette.git
````

2. Navigate to the project directory:

```bash
cd Musicbox_Roulette
```

3. Open Project File

4. Use in browser
   
## Usage

1. Click selected coin amount to put into the game.
2. Play game until your coins run out as you attempt to get a match.
